## 🎯 品牌定位公式

```
品牌：清微旅行
品類：清邁親子包車
差異化：台灣爸爸 Eric + 泰國媽媽 Min
```

### 核心原則
- **簡單** - 每個區塊只傳達一個概念
- **不重複** - 網站已有的詳細資訊，LINE 不重複（提供連結即可）
- **信任優先** - LINE OA 主要功能是加強信任，輔助顯示
- **對話導向** - 最終目標是引導客人開始對話

### 用戶旅程
```
部落格/網站 → 加入 LINE → 圖文選單（輔助信任）→ 開始對話
```
- 客人從網站導流過來，**已經對服務有基本認識**
- 圖文選單是**輔助**，有些人點、有些人不點
- 重點是**快速建立信任**，讓客人願意開口諮詢
