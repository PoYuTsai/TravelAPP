# 清微旅行首頁 / 包車頁 / About 頁 Brand Entity Brief

> **For Codex / Claude:** This brief translates the broader indexing strategy into page-level content requirements. Use it when editing homepage copy, `/services/car-charter`, or creating a future `/about` page.

**目標：** 讓 Google 更清楚理解：

- `清微旅行 Chiangway Travel` 是品牌
- `清邁親子包車` 是主要服務
- `Eric + Min` 是品牌真實人物與信任來源
- `親子家庭` 是核心受眾

**核心原則：**

- 不要為了 SEO 犧牲原本的品牌溫度
- 不要亂改整體前端設計
- 不要堆關鍵字
- 要讓品牌、服務、人物、地點四種訊號在每個重要頁都一致

---

## 1. 全站訊號規則

所有核心頁都應該自然出現以下四層訊號：

### 1.1 品牌訊號

- `清微旅行`
- `清微旅行 Chiangway Travel`

### 1.2 服務訊號

- `清邁親子包車`
- `清邁親子旅遊規劃`
- `清邁包車服務`

### 1.3 人物訊號

- `台灣爸爸 Eric`
- `泰國媽媽 Min`
- `住在清邁的在地家庭`

### 1.4 受眾訊號

- `親子家庭`
- `第一次帶孩子去清邁`
- `需要中文溝通與行程協助的家庭`

### 1.5 統一句型

各頁可以用不同語氣，但本質上都要能讀出這個句子：

`清微旅行 Chiangway Travel 是由台灣爸爸 Eric 與泰國媽媽 Min 在清邁經營的品牌，專做清邁親子包車與親子旅遊規劃。`

## 2. 首頁 Brief

### 2.1 首頁角色

首頁不是只有 landing page 功能，還必須是：

- 品牌定義頁
- Google 認品牌的主入口
- 其他頁面對品牌理解的母頁

### 2.2 Google 要從首頁學到什麼

首頁前 200 字內，應盡量讓 Google 理解這四件事：

1. 你是 `清微旅行`
2. 你做的是 `清邁親子包車`
3. 你是 `Eric + Min` 的在地家庭品牌
4. 你服務的是 `親子家庭`

### 2.3 不改版型前提下，首頁文案該補強的位置

#### Hero 區

目前 tone 可以保留，但建議附近文案一定要補明：

- 品牌名：`清微旅行`
- 主服務：`清邁親子包車`

**建議方向：**

- H1 可以維持溫度感
- subtitle 或 description 需要更明確定義品牌

**建議句型方向：**

- `清微旅行是由台灣爸爸 Eric 與泰國媽媽 Min 在清邁經營的品牌，專做清邁親子包車與親子旅遊規劃。`
- `如果你正在找適合親子家庭、可以中文溝通、又有在地照應的清邁包車服務，清微旅行就是我們現在在做的事。`

#### Who We Are 區

這一區是首頁非常重要的品牌實體強化區。

**必須回答的問題：**

- 你們是誰？
- 為什麼做這件事？
- 為什麼你們不是一般包車？

**必帶元素：**

- 台灣爸爸 / 泰國媽媽
- 在清邁生活
- 真實家庭
- 親子視角
- 司機與導遊分工

#### 首頁新增一個短區塊

建議未來加一個很短的品牌說明區塊，標題可以是：

- `清微旅行是什麼？`
- `我們做的不是一般清邁包車`

這區塊字數不用多，但要有強定義性。

### 2.4 首頁內鏈要求

首頁必須明確投票給下面兩頁：

- `/services/car-charter`
- `/tours`

建議 anchor：

- `清邁親子包車服務`
- `清邁親子行程案例`

不要首頁只把流量都導去 LINE，否則 Google 會覺得站內主題層級不夠清楚。

## 3. 包車頁 Brief

### 3.1 包車頁角色

`/services/car-charter` 是主商業頁，不是一般介紹頁。

這頁應該是你最主要承接下面這些查詢的 URL：

- 清邁親子包車
- 清邁包車
- 清邁中文包車
- 清邁親子旅遊包車

### 3.2 這頁要教 Google 的核心句

`清微旅行提供的是專為親子家庭設計的清邁親子包車服務，由 Eric 與 Min 在地經營，提供中文溝通、兒童座椅與司機導遊分工。`

### 3.3 首屏文案要求

首屏除了現在已有的賣點外，要更清楚把品牌與服務詞接起來。

**首屏至少要讓人一眼讀到：**

- `清微旅行`
- `清邁親子包車`
- `Eric + Min`
- `親子家庭`

### 3.4 正文區塊必帶的主題

這頁不只要講功能，要講「為什麼這個品牌可以做這件事」。

建議正文主軸：

1. 為什麼清邁親子包車需要親子視角
2. 清微旅行和一般包車的差別
3. 司機 / 導遊分工的價值
4. 中文溝通與行程照應
5. 兒童座椅 / 安全與彈性
6. 哪些家庭最適合你們

### 3.5 FAQ 應優先回答的商業問題

FAQ 不只是給使用者，也是在幫 Google理解服務邊界。

優先題型：

- 清邁親子包車怎麼預約？
- 清微旅行和一般清邁包車有什麼不同？
- 有提供兒童座椅嗎？
- 中文導遊和司機會一起嗎？
- 一日 / 多日怎麼安排？
- 適合第一次帶孩子去清邁嗎？

### 3.6 包車頁內鏈要求

這頁應該連出去：

- `/tours`
- `/contact`
- 1~2 篇最強的相關 blog

相關 blog 題材例如：

- 清邁交通
- 第一次去清邁怎麼安排
- 帶孩子去清邁適合去哪

## 4. About 頁 Brief

### 4.1 About 頁是否需要

需要。

原因不是因為使用者一定會大量看，而是：

- Google 很需要一個明確的品牌身份頁
- 使用者也需要一個「你們到底是誰」的信任頁

### 4.2 建議 URL

- `/about`

### 4.3 About 頁角色

這頁的主要任務不是成交，而是：

- 建立品牌實體
- 建立創辦人實體
- 建立品牌故事與服務之間的因果關係

### 4.4 About 頁建議內容結構

#### 第一段：一句話定義品牌

- `清微旅行 Chiangway Travel 是由台灣爸爸 Eric 與泰國媽媽 Min 在清邁經營的品牌，專做清邁親子包車與親子旅遊規劃。`

#### 第二段：我們為什麼做這件事

- 從家庭生活、孩子、在地資源、接待經驗出發

#### 第三段：我們服務的是誰

- 第一次來清邁的家庭
- 想要中文溝通的家庭
- 重視安全、節奏、彈性的家庭

#### 第四段：我們和一般包車有什麼差別

- 司機與導遊分工
- 親子視角
- 在地生活而不是單純帶景點

#### 第五段：官方聯絡與官方平台

- 官網
- LINE
- IG
- FB

### 4.5 About 頁要連到哪裡

- `/services/car-charter`
- `/tours`
- `/contact`
- 品牌故事相關 blog

## 5. Blog 內鏈規則簡版

### 5.1 每篇文章至少做三件事

1. 至少 1 次提到 `清微旅行`
2. 至少 1 個自然 anchor 指向 `/services/car-charter`
3. 至少 1 個自然 anchor 指向 `/tours` 或首頁

### 5.2 推薦 anchor

- `清邁親子包車`
- `清微旅行`
- `清邁親子行程規劃`
- `清邁親子旅遊安排`

### 5.3 文章類型與連結方向

- 交通 / 車程 / 接送類文章
  - 主連 `/services/car-charter`
- 行程 / 一日遊 / 多日遊類文章
  - 主連 `/tours`
- 品牌故事 / 生活感 / 在地觀察類文章
  - 主連首頁與未來 `/about`

### 5.4 不要做的事

- 不要每篇都同一句 CTA
- 不要每篇都只連 LINE
- 不要讓 blog 和商業頁像兩個網站

## 6. 實作優先順序

### 第一優先

- 首頁 Hero 附近的品牌定義句
- `/services/car-charter` 的品牌 + 服務對齊

### 第二優先

- About 頁規劃與落地

### 第三優先

- top 5 blog 文章內鏈回補

### 第四優先

- 依 Search Console 排查結果，逐步強化沒有被索引的重點頁

## 7. Claude / Codex Handoff

如果之後要讓 Claude 或 Codex 接著做，順序請固定為：

1. 先維持目前前端設計
