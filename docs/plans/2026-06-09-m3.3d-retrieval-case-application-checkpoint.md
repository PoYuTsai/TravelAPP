# M3.3d — Deterministic Retrieval-Case Application + Operator Preview

- 日期：2026-06-09
- Branch：`codex/line-oa-agent-mvp`
- 前置：M3.3c deterministic customer-change composer（`customer-itinerary-change-composer.ts`）
- 定位：把 `retrievalCases` 從被動白名單欄位，升級為**可代入 draft 的替代景點來源**，並加 operator preview。
- 硬邊界：仍**不**接 Notion live、**不**接 LLM、**不**接 LINE、**不**翻 gate；retrieval cases 只用測試 fixture 模擬。

## 範圍

- 不直接做 LLM refine / retrieval live / 翻 gate（刻意延後）。
- 只做 deterministic retrieval-case application + operator preview。

## 設計

### retrieval-case application（在 M3.3c decline 分支上擴充）

`CustomerChangeRequest.addActivities` 項目新增可選 `themeTag`。當某筆 add 因 limited
mobility 被判 unsuitable（比對共用 `MOBILITY_UNSUITABLE_TOKENS`）時，呼叫
`pickRetrievalAlternative(day, declinedActivity, themeTag, retrievalCases)`：

- 候選 = `retrievalCases` 中 `mobilityFriendly === true` 且**本身不在** unsuitable 表者。
- **substituted**：request 帶 `themeTag` 且候選有同 `themeTag` → 取第一筆（穩定排序）**代入 draft**
  （afternoonActivities），decision = `adjusted`，operatorNote 記來源（retrieval case + theme）。
- **named_only**：有 mobilityFriendly 候選但 theme 未對齊／未指定 → **不代入**，僅在
  customerExplanation 建議候選名，decision = `declined`（保留 M3.3c 行為，向後相容）。
- **none**：無可用候選 → 泛稱「較輕鬆的替代景點」，**絕不杜撰**景點名。

`customerExplanation` 維持客人乾淨版（禁 `themeTag`/`mobilityFriendly`/`retrieval`/`source`/
`候選`/`白名單`/`代入`/`內部` 等詞）。`operatorNotes` + 新 `retrievalApplications[]` 記錄替代來源與
outcome。lint error 仍 fail-closed（draft=null），但 `retrievalApplications` 仍回報。

### operator preview（新檔）

`customer-change-operator-preview.ts`：`buildOperatorRetrievalPreview(applications) → string[]`，
把每筆 `RetrievalApplication` 投影成 operator 出手前可讀的內部行（substituted 顯示已代入替代＋候選數；
named_only 列建議候選＋需人工挑；none 明確「請人工補景點，系統不杜撰」）。internal 詞允許（非客人版）。

## 契約（pinned by tests）

- themeTag match + mobilityFriendly → 代入 draft（adjusted）。
- candidate 非 mobilityFriendly → 不代入（none）。
- themeTag 有但無同 theme → named_only（建議未代入）。
- 無候選 → none，泛稱不杜撰。
- 多同 theme 候選 → 取第一筆（deterministic）。
- 代入後 render→lint 仍須過；lint error 仍 fail-closed 且 applications 仍回。
- customerExplanation 無 operator-only retrieval 詞。
