# 清微旅行 LINE 圖文選單正式規格

> **版本：2026-07-19 v2；狀態：Eric 已核准部署。**
> 本版取代舊的 2 × 3 選單提案，正式沿用目前的一大三小版型。圖文選單不公開完整「人數 × 地區 × 導遊」價目表。

## 1. 功能定位

客人已在 LINE 聊天室，不另設「開始詢價」按鈕。四格各自負責品牌、車況、產品與信任；點擊後以關鍵字回覆相關內容，客人可直接在同一聊天室接續提問。

```text
┌───────────────────────────────────────────────┐
│ 清微旅行                                      │
├───────────────┬───────────────┬───────────────┤
│ 車輛實拍      │ 招牌行程      │ 家庭實拍      │
└───────────────┴───────────────┴───────────────┘
```
