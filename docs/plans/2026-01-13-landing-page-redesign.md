# Landing Page 重新設計

## 品牌定位

**主標語**
> 清邁親子自由行

**副標語**
> 在地家庭經營，專為爸媽設計的旅程

**核心差異化**
- 台灣爸爸 + 泰國媽媽，真正住在清邁的家庭
- 自己有女兒，懂爸媽帶小孩出遊的需求

## 頁面結構

### 1. Hero 區塊

**目標**：3 秒內建立品牌印象

**內容**：
- 背景：一家三口在清邁的生活照/旅遊照
- 主標：清邁親子自由行
- 副標：在地家庭經營，專為爸媽設計的旅程
- 人物介紹：Eric & Min，住在清邁的台泰夫妻
- CTA：LINE 免費諮詢 ｜ 瀏覽行程

**設計重點**：
- 用真實家庭照取代庫存圖
- 人物資訊前置，建立信任感

**目標**：用數據建立可信度

**內容**：

| ⭐⭐⭐⭐⭐ | Google 五星好評（含連結） |

**資料來源**：Rezio 訂單記錄、Google 評論

---

### 3. 服務區塊

**目標**：展示一站式家庭旅遊服務

**內容**：

#### 親子包車服務
- 專屬司機中文溝通
- 兒童安全座椅
- 行程彈性不趕路
- 接機/送機服務
- CTA：查看行程方案

#### 芳縣特色民宿（Huen San Fang Hotel）
- 遠離觀光區的寧靜
- 體驗泰北在地生活
- 適合長住深度旅遊
- 民宿主人親自接待
- CTA：了解民宿詳情

**定位區隔**：
- 包車：短期旅遊（3-7天）
- 民宿：長住/深度旅遊
- 可搭配：包車玩市區 + 住民宿體驗鄉村

**注意**：不推其他飯店代訂，只推自家民宿

### 4. 為什麼選擇我們（差異化區塊）

**目標**：強化與競品的差異

**內容**：
1. **在地家庭經營** - 不是旅行社，是住在清邁的真實家庭
2. **自己也是爸媽** - 有女兒，懂帶小孩出遊的需求
3. **全程中文服務** - 從諮詢到旅途結束，溝通無障礙
4. **客製化行程** - 根據孩子年齡、體力量身打造

---

### 5. 精選文章區塊（SEO 入口）

**目標**：首頁導流到 SEO 文章，文章帶來自然流量

**內容**：
- 顯示 3 篇精選/最新文章
- 從 Sanity CMS 自動抓取 `featured: true` 的文章
- 卡片包含：封面圖、標題、摘要、分類標籤

---

### 6. 最終 CTA 區塊

**目標**：推動訪客行動

**內容**：
- 標題：準備好帶孩子來清邁了嗎？
- 副標：免費諮詢，讓在地爸媽幫你規劃最適合的親子行程
- CTA：LINE 免費諮詢 ｜ 瀏覽行程

## 前後端對應（Sanity CMS）

### 文章編輯欄位 → 前端顯示

| 後端欄位 | 你做的事 | 前端顯示位置 |
|----------|----------|--------------|
| 文章標題 | 輸入文字 | H1 標題 + SEO title |
| 網址代碼 | 自動產生/自訂 | URL slug |
| 封面圖片 | 上傳 + 填 alt | 文章頂部大圖 |
| 文章摘要 | 80-160 字 | 列表卡片 + 社群分享 |
| 文章內容 | 富文本編輯 | 自動排版 |
| ├ H2/H3 | 選樣式 | 自動產生目錄 |
| ├ 圖片 | 上傳 + 說明 | 文內圖 + caption |
| ├ LINE CTA | 點擊插入 | 黃色諮詢區塊 |
| ├ 推薦行程 | 選擇行程 | 行程卡片連結 |
| └ 提示框 | 選類型 + 內容 | 彩色提示框 |
| SEO 描述 | 120-160 字 | Google 搜尋結果 |
| 分類 | 選擇 | 文章標籤 |
| 精選文章 | 開關 | 首頁/列表頁置頂 |
| 發佈日期 | 選擇 | 文章日期顯示 |
| 最後更新 | 選擇 | SEO 更新標記 |
