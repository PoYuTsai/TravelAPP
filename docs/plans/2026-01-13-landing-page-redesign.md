## 1. 目標與成功指標

### Main Goal（主要目標）

透過 SEO 內容行銷建立自然流量，取代付費廣告，將精準客戶導入 LINE OA 完成包車預訂。

### 核心漏斗

```
SEO 文章（AI + 真實經驗）
        ↓
   Google 自然流量
        ↓
   Landing Page / 部落格
        ↓
   LINE OA 諮詢
        ↓
   包車導遊預定
```

### 商業目標

| 指標 | 目標值 |
|------|--------|
| 月利潤 | 20 萬 TWD |
| 月訂單數 | 15 組 |
| 訂單條件 | 親子團、有導遊、4天+、含大象營 |
| 流量來源 | 從廣告 → SEO 自然流量 |

### SEO 指標（6-12 個月）

| 指標 | 目標 |
|------|------|
| 「清邁親子包車」排名 | Google 前 10 |
| 月 UV | 3,000+ |
| LINE 諮詢 | 30+/月（轉換率 50% = 15 單）|

### 內容產出目標

| 階段 | 頻率 |
|------|------|
| 網站完成前 | 規劃關鍵字 + 文章結構 |
| 網站完成後 | 2 天 1 篇（15 篇/月）|

### 時間預期

- SEO 是長期資產投資
- 預期 6-12 個月開始看到明顯效果
- 持續產出、持續優化

## 2. 品牌定位

### 主標語
> **清邁親子自由行**

### 副標語
> **在地家庭經營，專為爸媽設計的旅程**

### 核心差異化
- 台灣爸爸 + 泰國媽媽，真正住在清邁的家庭
- 自己有女兒，懂爸媽帶小孩出遊的需求
- 不是旅行社，是家庭經營的溫度服務
- 包車 + 自家民宿，一站式服務
- 司機 + 導遊專業分工（非中文司機一人包辦）

### 目標客群
- 台灣家庭（有小孩）
- 計劃清邁親子自由行
- 需要中文服務
- 重視安全、彈性、溫度

### 定位策略

> **親子定位 = 精準篩選器**
> 打親子 → 吸引重視品質的家庭 → 口碑擴散到其他族群（年輕人、長輩）

- 品類專注：只做泰國清邁
- 定位打法：最精而深（非廣而淺）

## 3. 關鍵字策略與內容規劃

### 關鍵字分層

| 優先級 | 類型 | 範例關鍵字 | 文章數 |
|--------|------|-----------|--------|
| ⭐⭐⭐ | 服務型（主攻）| 清邁親子包車、清邁包車推薦、清邁包車一日遊 | 5-8 篇 |
| ⭐⭐ | 攻略型 | 清邁親子景點、清邁親子行程、清邁帶小孩幾天 | 10-15 篇 |
| ⭐⭐ | 痛點型 | 清邁親子餐廳、大象保育園推薦、清邁嬰兒推車 | 10-15 篇 |
| ⭐ | 長尾型 | 清邁親子五天四夜、清邁大象營比較、清邁雨季帶小孩 | 持續擴充 |

### 內容差異化原則

每篇文章必須包含：
1. **真實經驗** - 「我們帶女兒去過...」的第一人稱視角
2. **親子角度** - 就算是通用景點，也從爸媽視角切入
3. **實用價值** - 解決一個具體問題，不寫空泛介紹
4. **轉換鉤子** - 自然帶入「需要包車可以問我們」

### 文章產出流程

```
1. 選題（根據關鍵字清單）
      ↓
2. AI 生成大綱 + 初稿
      ↓
3. Eric 加入真實經驗、修改語氣
      ↓
4. AI 檢查 SEO（標題、meta、內連結）
      ↓
5. 發布到 Sanity CMS
      ↓
6. 分享到社群（FB/IG）擴大曝光
```

## 4. 核心賣點與訊息策略

### 三大護城河（別人難複製）

| 賣點 | 一句話說明 | 內容呈現方式 |
|------|-----------|-------------|
| **在地家庭** | 台灣爸爸 + 泰國媽媽，真正住在清邁 | 家庭照、生活故事、在地視角 |
| **自己是爸媽** | 有女兒，懂帶小孩出遊的眉角 | 「我們帶女兒...」的真實經驗 |
| **自家民宿** | 包車 + 芳縣民宿，一站式服務 | 民宿頁面、深度旅遊路線 |

### 差異化定位：司機 + 導遊分開

| 一般同行 | 清微旅行 |
|----------|----------|
| 中文司機（一人包辦） | 司機專心開車 + 導遊專心服務 |
| 開車時無法講解 | 全程有人陪伴解說 |
| 司機疲勞影響安全 | 專業分工，各司其職 |

> 這是文章中可以強調的點：「為什麼我們堅持司機導遊分開？」

### 價格呈現策略

| 位置 | 呈現方式 |
|------|----------|
| Landing Page | 「每日 NT$ X,XXX 起」區間價 |
| 服務頁面 | 不同車型/天數的起始價格表 |
| 文章內 | 不直接寫價格，導向 LINE 諮詢 |
| LINE 諮詢 | 根據需求客製報價 |

### 品牌語調

- ✅ 像朋友聊天，不像旅行社話術
- ✅ 用「我們」「女兒」拉近距離
- ✅ 分享真實經驗，包含踩雷的
- ❌ 不用「最專業」「最優質」等空泛詞
- ❌ 不貶低同行，只說自己的特色

## 7. SEO 技術規格

### URL 結構

| 頁面類型 | URL 格式 | 範例 |
|----------|----------|------|
| 首頁 | `/` | chiangway.com |
| 服務頁 | `/services/[slug]` | `/services/car-charter` |
| 民宿頁 | `/homestay` | `/homestay` |
| 部落格列表 | `/blog` | `/blog` |
| 文章頁 | `/blog/[slug]` | `/blog/chiang-mai-elephant-sanctuary` |
| 關於我們 | `/about` | `/about` |

### Meta Tags 規範

```
首頁：
- Title: 清邁親子包車 | 清微旅行 - 在地家庭經營
- Description: 台泰夫妻在地經營，專為親子家庭設計的清邁包車服務。司機導遊專業分工，行程彈性不趕路。LINE 免費諮詢。

文章頁：
- Title: [文章標題] | 清微旅行
- Description: [文章摘要，150字內，含關鍵字]
```

### Schema Markup（結構化資料）

| 頁面 | Schema 類型 | 用途 |
|------|-------------|------|
| 全站 | `LocalBusiness` | 顯示商家資訊、評價星星 |
| 文章 | `Article` + `BreadcrumbList` | 文章搜尋結果優化 |
| 服務頁 | `Service` | 服務項目顯示 |
| FAQ | `FAQPage` | 常見問題直接顯示在搜尋結果 |

### 內部連結策略

```
首頁
  ├── 服務頁（包車）──→ 相關文章
  ├── 民宿頁 ──→ 芳縣深度遊文章
  └── 部落格
        ├── 攻略文 ──→ 服務頁 CTA
        ├── 景點文 ──→ 相關景點文
        └── 痛點文 ──→ 服務頁 CTA

每篇文章至少：
- 1 個連到服務頁（轉換）
- 2-3 個連到相關文章（停留時間）
```

### 圖片 SEO

| 項目 | 規範 |
|------|------|
| 檔名 | 用英文描述，如 `chiang-mai-elephant-sanctuary.jpg` |
| Alt 文字 | 描述圖片內容 + 關鍵字，如「清邁大象保育園親子體驗」|
| 格式 | WebP 優先，fallback JPG |
| 壓縮 | 控制在 200KB 以內 |

## 8. Mobile-First 與效能規格

### 流量來源預估

| 裝置 | 預估比例 | 原因 |
|------|----------|------|
| 手機 | 70-80% | 爸媽用手機查資料、LINE 諮詢 |
| 桌機 | 15-25% | 規劃行程時用電腦 |
| 平板 | 5% | 少數 |

> 設計以 Mobile-First 為主，桌機是放大版

### 手機版設計規範

| 元素 | 規格 |
|------|------|
| 觸控區域 | 最小 44x44px（拇指友善）|
| 字體大小 | 內文最小 16px（避免縮放）|
| CTA 按鈕 | 全寬或至少 48px 高 |
| LINE 浮動按鈕 | 固定右下角，隨時可點 |
| 導覽列 | 漢堡選單，Logo 置中 |
| 圖片 | 自適應寬度，保持比例 |

### 效能目標（Core Web Vitals）

| 指標 | 目標 | 說明 |
|------|------|------|
| LCP | < 2.5s | 最大內容繪製（Hero 圖片載入）|
| INP | < 200ms | 互動回應速度 |
| CLS | < 0.1 | 版面穩定度（圖片不跳動）|

### 效能優化策略

| 項目 | 做法 |
|------|------|
| 圖片 | Next.js Image 自動優化 + WebP + lazy loading |
| 字體 | `next/font` 載入 Noto Sans TC，避免 FOUT |
| CSS | Tailwind CSS purge 未使用樣式 |
| JS | 動態載入非首屏元件 |
| 快取 | 靜態頁面 ISR，文章更新時 revalidate |

### 載入體驗

| 狀態 | 處理方式 |
|------|----------|
| 圖片載入中 | Blur placeholder（模糊佔位圖）|
| 頁面切換 | 輕量 loading indicator |
| 文章列表 | Skeleton 骨架屏 |

## 9. 無障礙規範

### 基本規範（WCAG 2.1 AA）

| 項目 | 規範 | 做法 |
|------|------|------|
| 色彩對比 | 文字至少 4.5:1 | 深灰 `#374151` 配白底 ✅ |
| 鍵盤操作 | 所有互動元素可用鍵盤 | Focus 樣式明顯可見 |
| 圖片替代文字 | 所有圖片有 alt | 描述內容，非裝飾用空 alt |
| 表單標籤 | 所有輸入有 label | 搜尋框、訂閱表單等 |
| 連結文字 | 避免「點此」「更多」 | 用「查看包車服務」等明確文字 |

> 不需要做到 100% 無障礙，但基本項目要顧到，對 SEO 也有幫助

## 11. 競品差異化策略

### vs 清邁廢式

| 面向 | 清邁廢式 | 清微旅行 | 差異化策略 |
|------|----------|----------|-----------|
| 品牌形象 | 專業旅遊媒體人 | 在地爸媽家庭 | 強調「真實家庭」溫度 |
| 目標客群 | 全客群 | 專注親子家庭 | 內容 100% 圍繞親子 |
