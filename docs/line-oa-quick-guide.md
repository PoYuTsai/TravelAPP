## 4. 執行 Checklist

### Step 1：圖片設計
- [ ] 給設計師上方規格
- [ ] 收到完稿圖片

### Step 2：LINE 後台設定
- [ ] 選擇「1大 + 3小」版型
- [ ] 上傳完稿圖片
- [ ] A 區塊 → 文字訊息（貼上內容）
- [ ] B 區塊 → 卡片輪播（設定 2 張卡片）
- [ ] C 區塊 → 文字訊息（貼上內容）
- [ ] D 區塊 → 文字訊息 + 影片

### Step 3：關鍵字設定
- [ ] 新增「VIP 價格」關鍵字
- [ ] 新增「小車價格」關鍵字
- [ ] 刪除所有舊關鍵字

### Step 4：測試
- [ ] 點擊每個區塊確認內容正確
- [ ] 輸入「VIP 價格」測試
- [ ] 輸入「小車價格」測試
